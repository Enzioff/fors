Java-разработчик в ФОРС – Центр разработки: Обязанности, Требования и Условия работы


В отделении разработки информационных систем ФОРС – Центр разработки открыта позиция: java-разработчик. Мы ищем специалиста с опытом в проектировании и разработке масштабируемых веб-приложений. Ваша работа будет связана с созданием высоконадежных решений для корпоративных клиентов, включая внедрение инновационных технологий и оптимизацию бизнес-процессов.



Ключевые обязанности Java-разработчика в ФОРС – Центр разработки


Роль java-разработчика в нашей компании охватывает полный цикл разработки серверной части систем, устойчивых к отказам и высоким нагрузкам. Основные задачи включают:


  • Разработка на Java серверной части системы, устойчивой к отказам и высоким нагрузкам согласно формальным заданиям
  • Создание моделей данных, служб, контроллеров, представлений, первичное тестирование и документирование кода
  • Рефакторинг и оптимизация кода
  • Написание юнит-тестов
  • Участие в кросс-ревью
  • Участие в подготовке проектной документации
  • Участие в проектировании архитектуры приложения


Требования к кандидату на должность java-разработчика


Для того чтобы успешно выполнять вышеописанные задачи, кандидат на позицию должен обладать рядом профессиональных качеств и навыков, включая:


  • Высшее профильное образование
  • Отличное значение Java 11 и выше
  • Опыт разработки на Java более 3х лет
  • Опыт работы со Spring (Core, Boot, MVC, Security, JDBC, JPA, Cloud)
  • Опыт написания простых SQL запросов
  • Опыт написания автоматизированных тестов
  • Опыт работы с Maven, GIT, Linux, JIRA


Работа в ФОРС – Центре разработки предлагает целый ряд преимуществ, среди которых:


  • «Белая» заработная плата, оформление по ТК РФ
  • ФОРС – Центр разработки является аккредитованной ИТ-компанией
  • ДМС, НС — после испытательного срока (3 месяца) + страховка выезда за рубеж (полис ВЗР)
  • Техника, по выбору
  • Возможность профессионального и карьерного развития, участие в конференциях
  • Дополнительный выходной день в честь дня рождения
  • Комфортные условия труда
